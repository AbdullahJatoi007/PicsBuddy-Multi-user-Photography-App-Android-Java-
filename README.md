# PicsBuddy

📱 **Overview**
PicsBuddy is a multi-user photography app similar to Pexels. Users can upload, browse, and interact with photos. Admin and moderation features are integrated.

🎯 **Problem it Solves**
Allows photographers to showcase their work, interact with users, and manage content securely.

✨ **Features**
- User registration & login
- Multi-user photo upload & browsing
- Push notifications
- Report/block users
- Like & comment functionality
- Admin moderation features

🛠 **Tech Stack**
- Android (Java, XML)
- Firebase Authentication & Firestore
- Firebase Cloud Messaging (Push Notifications)

🧠 **Architecture**
- Clean separation of UI, backend calls, and notifications
- Multi-user content management system
